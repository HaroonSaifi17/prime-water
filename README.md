# Prime Water

Prime Water is a modern, high-conversion B2B/B2C platform built with SvelteKit and Tailwind CSS. 

## Features
- **Nano Banao AI Studio:** A sophisticated, component-driven UI for users to upload their brand logos and configure custom water bottle labels.
- **Premium Design:** "The Clean Slate" aesthetic focusing on high contrast, crisp typography, and stunning glassmorphic UI elements.
- **Smooth Animations:** Native IntersectionObserver scroll reveals for a premium feel without sacrificing performance.
- **Fully Componentized:** Clean architecture separating the Hero, Process, and Studio sections for high maintainability.

## Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Structure
- `src/lib/components/layout/` - Global layout components (Navbar, Footer)
- `src/lib/components/home/` - Homepage sections (Hero, Process, Studio)
- `src/routes/` - Main page layouts
